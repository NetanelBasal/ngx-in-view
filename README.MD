## Is Element In Viewport

### Installation

To install this library, run:

```bash
$ npm install ngx-in-view --save
```
```typescript
import { InViewModule } from 'ngx-in-view';

@NgModule({
  imports: [
    InViewModule.forRoot()
  ]
})
export class AppModule { }
```
